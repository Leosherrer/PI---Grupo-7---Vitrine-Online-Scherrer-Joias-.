Projeto Vitrine Online
O Nosso grupo vai desenvolver um sistema chamado "Vitrine Online" que vai permitir aos clientes acessarem todo o catálogo de produtos, ver descrições detalhadas dos produtos, dividir os produtos em categorias (ex: Braceletes, Anéis, Brincos etc.), mostrar preços, permitir compras, retirar na loja, enviar entregadores, conversa entre clientes e atendentes, interface intuitiva e fácil uso.
